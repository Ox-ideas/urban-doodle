# Local LLM

An LLM running on the user's local hardware (e.g., via [[Olama]] or LM Studio) rather than a cloud provider.

## Advantages for LLM Wiki
- **Privacy**: No data leaves the machine.
- **Cost**: Free to run after hardware purchase.
- **Safety**: Can be combined with an [[Agentic Firewall]] for a fully "air-gapped" knowledge system.
