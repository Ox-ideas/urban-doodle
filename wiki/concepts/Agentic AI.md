Agentic AI refers to the automation layer that performs ingestion, maintenance, querying, and validation in the LLM Wiki.

## Role
- detect new raw sources
- extract concepts and create wiki pages
- keep the wiki consistent with schema rules
- run maintenance, lint, and query workflows

## Key capabilities
- vault-safe access control for Obsidian operations
- script-driven processing for repeatability
- human-in-the-loop review when using lower-confidence models
