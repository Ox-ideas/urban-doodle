# Idea File

A conceptual format introduced by [[Andrej Karpathy]] for the era of AI agents. Instead of sharing specific code or applications, an "Idea File" communicates a high-level pattern or architecture.

The user's agent then takes this abstract idea and instantiates it into a concrete build customized to the user's specific environment, tools, and preferences. This makes ideas more portable and less prone to bit-rot than specific codebases.
