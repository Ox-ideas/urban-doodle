A knowledge graph is a structured network of notes and relationships that improves the way AI understands and retrieves knowledge.

## Purpose
A knowledge graph makes explicit how concepts, entities, sources, and decisions connect. This is especially useful for AI because it can use the structure to avoid reading irrelevant material and to trace reasoning more reliably.

## Edge semantics
Typed edges can indicate relationships such as:
- supports
- contradicts
- depends on
- derived from
- authored
- related to

These richer edge types allow the graph to express meaningful connections rather than generic links.

## AI-centric retrieval
By keeping notes atomic and explicit, the knowledge graph reduces token waste and makes it easier for an agent to find the right page for a question.
