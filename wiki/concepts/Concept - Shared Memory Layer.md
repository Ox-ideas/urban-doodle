# Shared Memory Layer

A structured system that allows AI agents and humans to access a consistent, evolving pool of knowledge. It aligns with how the user thinks and prevents knowledge loss between different tools.

In the context of an [[LLM Wiki]], the wiki itself acts as this layer, bridging the gap between raw data and actionable insight.
