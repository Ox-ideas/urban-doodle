Query is the process of using the wiki to answer questions and generate insights.

## Workflow
- search `wiki/index.md` for relevant pages
- read the identified pages and their references
- synthesize an answer with wikilink citations
- optionally file the answer as a new wiki page or update an existing page

## Best practices
- do not invent unsupported citations
- use the wiki's structured pages rather than raw sources where possible
- capture valuable outputs back into the wiki so knowledge compounds
