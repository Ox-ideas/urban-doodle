## Summary
This source argues that traditional second brain PKM structures are suboptimal for AI and proposes an "Infinite Brain" approach. It emphasizes more atomic note types, richer edge semantics, and AI-centric organization to reduce token waste and improve retrieval quality.

## Key points
- Traditional PARA-style second brain systems produce large notes and untyped links that are inefficient for AI.
- Infinite Brain uses 16 note types and atomic summaries to make content easier for AI to retrieve and reason over.
- Edge semantics such as supports, contradicts, depends on, derived from, and authored improve knowledge graph quality.
- The system is intended to make notes more useful for AI while still supporting human understanding.
