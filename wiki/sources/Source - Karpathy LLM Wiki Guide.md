# Source: Karpathy's LLM Wiki: The Complete Guide

A deep dive by [[Antigravity.codes]] into the "Idea File" shared by [[Andrej Karpathy]] regarding the [[LLM Wiki]] pattern.

## Key Takeaways
- **Idea Files**: A new format for the agent era where one shares the "idea" (pattern) rather than specific code, letting the user's agent customize the build.
- **Wiki vs RAG**: Traditional RAG rediscovery knowledge per query; a Wiki compiles it once and compounds it.
- **Three-Layer Architecture**: Confirms the Raw/Wiki/Schema structure as the standard for this pattern.
- **Tooling**: Mentions [[qmd]] for local search and [[Obsidian Web Clipper]] for ingestion.
- **Historical Context**: Links the idea back to Vannevar Bush's **Memex** (1945).
