# Source: How To Build LLM Wiki In Obsidian?

A practical guide by [[Wanderloots]] on building a "Shared Memory Layer" for AI agents using Obsidian.

## Key Takeaways
- **Three-Layer Architecture**: Sources (immutable), Wiki (LLM-owned), and Schema (the rules).
- **Agentic Firewall**: A security wrapper to prevent agents from accessing unauthorized vaults.
- **Git Integration**: Using version control as "save points" for knowledge.
- **Local LLMs**: Using tools like Olama to run the wiki privately and for free.
- **Vibe Coding**: The agent builds the specifics based on high-level patterns in the schema.
