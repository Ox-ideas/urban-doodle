Obsidian CLI is the tool that enables agentic automation to interact with the Obsidian vault.

## Role
- allows scripts and agents to read and write vault files
- supports vault-safe wrappers to restrict operations to the correct folder
- enables programmatic ingestion and maintenance workflows
